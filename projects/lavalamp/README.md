# Lavalamp Baseline

This directory is the public-safe Skeleton baseline for `alanua/Lavalamp`.
It records project identity, current handoff state, and source placeholders only.

## Scope

- WLED transparent cylinder lava lamp project.
- Separate from Skeleton core, Jeeves runtime, and general home automation.
- Public-safe docs and placeholders are allowed here.
- Real firmware, effect implementations, device configuration, and hardware-specific notes require review before they are promoted into public source.

## Exclusions

The baseline must not contain:

- binaries or firmware images;
- PlatformIO, WLED, or compiler build cache;
- OTA payloads or OTA configuration;
- secrets, credentials, tokens, Wi-Fi details, or environment files;
- private device addresses, private paths, screenshots, or logs.

## Source Routes

- `projects/lavalamp/src/README.md` reserves the future source route for reviewed effect or usermod code.
- `projects/lavalamp/firmware/README.md` reserves the future firmware route for reviewed build instructions and source references.

Until reviewed source exists, these routes remain placeholders.
